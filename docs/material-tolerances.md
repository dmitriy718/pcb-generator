# Material Tolerance Guide

Material profiles drive default clearances, snap-fit spacing, hole compensation,
minimum feature size, wall thickness, layer height, infill, and Bambu profile hints.

These defaults are conservative FDM starting points:

- PLA: tight clearances, general prototypes.
- PETG: slightly larger clearances for stringing and flexibility.
- ABS / ASA: larger clearances for shrinkage and enclosed printing.
- TPU: larger spacing and thicker features for flexible prints.
- Carbon Fiber PLA: hardened-nozzle profile with modest clearance increase.
- Nylon: dried-filament profile with larger clearance for shrinkage and toughness.

Always validate critical screw, insert, magnet-pocket, and snap-fit dimensions with a
small calibration coupon before production batches.
